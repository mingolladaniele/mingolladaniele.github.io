ARTICLEE
ARTICLEE is a web application built with Python and based on the Flask micro-framework. It enables users to share their newspaper articles with the community. Key features include:
  • User login and registration system;
  • Creation, editing, and deletion of articles;
  • Fully responsive design (Bootstrap 4);
  • Compatible with multiple databases (SQLAlchemy);
  • Integration of Stripe API for seamless transactions.
