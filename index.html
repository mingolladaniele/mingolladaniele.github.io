Parallel pipeline for data analysis
Parallel pipeline entirely developed in Python for the collection, integration and analysis of data coming from the online catalogs of the main Italian publishing houses. The aim was to determining the characteristics that most influence the price of a book. The distributed data storage is done through Hadoop while their parallel processing is done through Dask.

ARTICLEE
Articlee is a web application developed in Python and based on the Flask micro-framework. It allows users to share their newspaper articles with the community. Main features:
  • user login and registration system;
  • possibility to create, edit or delete articles;
  • fully responsive (Bootstrap 4);
  • compatible with multiple databases (SQLAlchemy);
  • Stripe API support.
