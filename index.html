Progetti

Dedico molto del mio tempo nella realizzazione e nel perseguimento di un percorso di studi personalizzato parallelo a quello universitario, interamente realizzato da me. Oltre all'approfondimento di materie prettamente tecniche, il percorso comprendo lo studio della filosofia e del pensiero critico. La mia formazione si basa sulla lettura di libri e articoli scientifici relativi agli ambiti di mio interesse, sulla frequentazione di corsi online e sul costante dialogo con esperti di ogni settore (ascolta il mio podcast).

Pipeline distribuita per l'analisi dei dati
Pipeline distribuita interamente sviluppata in Python per la raccolta, l’integrazione e l’analisi dei dati provenienti dai cataloghi online delle principali case editrici italiane con il fine di determinare le caratteristiche che maggiormente influenzano il prezzo di un libro. La memorizzazione distribuita dei dati avviene tramite Hadoop mentre il loro processamento parallelizzato tramite Dask.

ARTICLEE
Articlee è un'applicazione web sviluppata in Python e basata sul micro-framework Flask. Consente agli utenti di condividere i propri articoli di giornale con la community. Principali caratteristiche:
  • sistema di login e registrazione degli utenti;
  • possibilità di creare, modificare o eliminare gli articoli;
  • completamente responsive (Bootstrap 4);
  • compatibile con molteplici database (SQLAlchemy);
  • supporto API Stripe.
