ARTICLEE
Articlee è un'applicazione web sviluppata in Python e basata sul micro-framework Flask. Consente agli utenti di condividere i propri articoli di giornale con la community. Principali caratteristiche:
  • Sistema di login e registrazione degli utenti;
  • Possibilità di creare, modificare o eliminare gli articoli;
  • Completamente responsive (Bootstrap 4);
  • Compatibile con molteplici database (SQLAlchemy);
  • Supporto API Stripe;
